# Funcionalidade Register
Este projeto foi desenvolvido a partir de uma atividade da disciplina de Arquitetura de Software como o intuito de incentivar os alunos a aplicarem as arquiteturas ensinadas em sala de aula . A arquitetura escolhida foi microserviços e foi feita a partir do caso de uso de um sistema de gerenciamento escolar. Fui responsável por desenvolver a funcionalidade de Cadastro de alunos, professores e disciplinas.

# Tecnologias 
 - Java 21
 - Spring Boot 4
 - PostgreSQL

# Habilidades Desenvolvidas
  - Conhecimento prático em Java
  - Padrões de projeto (clean code)
  - Conhecimento prático de arquitetura de microserviços
  - Desenvolvimento e consumo de API Rest
  - Conhecimento prático em banco de dados relacionais
  - Conhecimento prático em SQL e consultas simples
  - Conhecimento prático no framework Spring Boot
  - Conhecimento prático com versionamento utilizando Git
